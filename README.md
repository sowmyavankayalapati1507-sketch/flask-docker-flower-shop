#  Flower Bouquet Shop (Flask + Docker)

A full-stack web application to browse and order flower bouquets, built using Flask and deployed using Docker and Render.

---

##  Features

* Display flower bouquets with images
* Add products to cart
* Calculate total price dynamically
* Simple and responsive UI

---

##  Tech Stack

* Python (Flask)
* HTML, CSS, JavaScript
* Docker
* Render (Deployment)

---

##  Docker Setup

Build Docker image:
docker build -t flower-shop .

Run container:
docker run -p 5000:5000 flower-shop
